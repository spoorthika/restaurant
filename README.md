# Restaurant App

This is a simple restaurant project made using Python Flask, MySQL, HTML, CSS, and JavaScript.

## Features

- View orders
- View payments
- View menu
- Displays data in table format on a web page

## Files

- `restaurant.py` -backend
- `index.html` - Frontend page
- `restaurant.sql` - Database structure and data
- CSV files - Used for importing table data into MySQL

## Tools Used

- Python
- Flask
- MySQL
- HTML
- CSS
- JavaScript

## How to Run

1. Open MySQL and create the `restaurant` database.
2. Import the SQL file and CSV files into MySQL.
3. Open terminal and go to the project folder.
4. Run:

```bash
py restaurant.py
```

5. Enter your MySQL password.
6. Open in browser:

```
http://127.0.0.1:5000/
```

## Output

The project shows:
- Orders table
- Payments table
- Menu table
